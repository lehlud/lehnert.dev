Hello to whoever is reading this!

This blog entry is mainly for testing purposes.

Here's a link to Google, try clicking on it!

If you haven't noticed so already, this blog is heavily inspired by handwritten.blog. I've discovered Daniel's blog last week and almost instantly thought about how I could create my own handwritten blog. I don't know if Daniel will ever read this, but if you do, please keep up the work, I am a fan!

And just like Daniel, I will only accept comments that have been written by hand and mailed to blog@lehnert.dev.

But unlike handwritten.blog, this blog is brought to you by Simple Vector Graphics (SVG)s instead of Portable Network Graphics (PNG)s. This comes at the disadvantage of bigger file sizes (currently, this blog entry weighs in at 25 MB without optimizations and still 1.8 MB when optimized using Inkscape on default settings). However, I am convinced (and optimistic) to further reduce the size of handwritten SVGs in the future.
